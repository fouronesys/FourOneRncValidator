Endpoints Disponibles

Principales endpoints de nuestra API

GET /api/validate/{rnc}
Validar RNC

Verifica si un RNC existe en la base de datos y devuelve su estado.

GET /api/info/{rnc}
Información Completa

Obtiene toda la información disponible para un RNC específico.

POST /api/search
Búsqueda por Lotes

Valida múltiples RNCs en una sola petición (máximo 10).

GET /api/search-by-name
Búsqueda por Nombre

Busca empresas por nombre con sugerencias mientras escribes.

GET /api/status
Estado de la API

Información sobre el estado de la API y estadísticas de la base de datos.
